Markov Jump Linear Systems (MJLS) are linear dynamical systems whose parameters are governed by a Markov chain $[\theta(k),k\geq 0]$.
The term "hidden" refers to the fact that the parameter $\theta$ is indirectly observed via a related process $[\eta(k),k\geq 0\]$.

The proposed algorithm generates examples of Hidden Markov Jump Linear Systems (HMJLSs) with parameters driven (as suggested) by a hidden Markov chain.
